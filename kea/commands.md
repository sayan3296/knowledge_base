# Commands
## Kea Control Agent
Validate config
```shell
# As root:
/usr/sbin/kea-ctrl-agent -t /home/lstejska/projects/kea/conf/kea-ctrl-agent.json
```

Run
```shell
# As root:
/usr/sbin/kea-ctrl-agent -c /home/lstejska/projects/kea/conf/kea-ctrl-agent.json
```
